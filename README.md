# 키보드 입력 테스트 (Keyboard Input Test)

브라우저에서 키보드 입력을 실시간으로 시각화하는 간단한 웹 애플리케이션입니다.
물리 키보드를 누르거나 화면의 키를 클릭하면 해당 키가 하이라이트됩니다.

## 주요 기능

- **실시간 하이라이트** — 키를 누르면 화면 키보드의 해당 키가 표시됩니다.
- **클릭 / 터치 입력** — 마우스 클릭이나 터치로도 동일하게 동작합니다.
- **입력 정보 패널** — `event.key`, `event.code`, `keyCode`, 키 위치(왼쪽/오른쪽/넘패드)를 표시합니다.
- **테스트 진행률** — 한 번이라도 누른 키를 추적해 진행 바로 보여줍니다.
- **초기화** — 테스트 기록을 리셋합니다.
- **다크 모드** — 우측 상단 토글로 라이트/다크 테마를 전환합니다. 선택한 테마는 `localStorage`에 저장되어 다음 방문 시 유지되며, 별도 선택이 없으면 시스템 설정(`prefers-color-scheme`)을 따릅니다.

## 사용 방법

별도 설치나 빌드 없이 `index.html`을 브라우저로 열면 됩니다.

```bash
xdg-open index.html   # Linux
open index.html       # macOS
```

## 구성

- `index.html` — HTML/CSS/JavaScript가 모두 포함된 단일 파일 (외부 의존성 없음)

## 디자인

심플한 모노크롬 스타일, TKL(텐키리스) 키보드 레이아웃. 라이트/다크 두 가지 테마를 지원합니다.

## 참고

- `F5`, `F11`, `F12` 등 일부 키는 브라우저/OS 레벨 동작이라 화면에 반영되지 않을 수 있습니다.
- 넘패드 키는 화면 레이아웃에는 없지만 정보 패널에는 정상 표시됩니다.
